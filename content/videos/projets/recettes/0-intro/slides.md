# Préparations

- Concept et projet
- Méthodologie
- Outils utilisés

## Concept

- Projet de A à Z
- Utilisation de plusieurs technos et outils
- Explications au fur et à mesure, depuis 0

## Connaissances préalables

- Bases de programmation
- Programmation Orientée Objet (POO)
- Git / Gestion de version

## Quel projet ?

Un site de recettes pour :

- chercher des recettes, 
- filtrer selon des restrictions (végé, végan, etc.)
- filtrer selon des ingrédients
- utilisable sur mobile
- application mobile à l'avenir

## Technos

- Back : Symfony + API Platform
- Front : VueJS + Quasar

## Méthodologie

- Présentation 
  + des problèmes rencontrés
  + des solutions trouvées 
  + une partie du code en direct
- Code disponible sur GitHub
- Une vidéo par quinzaine / par mois
- Suivi de l'avancement sur Trello

## Outils abordés

- Trello
- Git + GitHub
- Toggl
- CI/CD avec GitHub Actions
- Symfony + API Platform
- VueJS + Quasar
- Docker et Docker-compose

## Trello

Un [tableau pour le projet](https://trello.com/b/i54gZUTD/recettes-projet-sf-vuejs) pour :
- voir les tâches à faire
- leur avancement
- les priorités

## Git + GitHub

[Git](https://git-scm.com/) pour versionner le code.
[GitHub](https://github.com/) pour le travail collaboratif :
- "sauvegarde" en ligne
- relectures du code produit
- fusion 
- Mise en ligne

:warning: Un ticket Trello = une branche

## Toggl

[Outil de mesure du temps de travail](https://track.toggl.com/timer)
- des projets
- des tags
- des statistiques

## CI/CD avec GitHub Actions

Automatiser les tâches répétitives
- vérifier que le code "compile"
- lancer les tests
- vérifier la qualité du code
- déployer le site
- etc.

## Symfony + API Platform

[Symfony 6](https://symfony.com/) comme framework de base
[API Platform](https://api-platform.com/) pour créer rapidement une API

## VueJS + Quasar

[VueJS 3](https://vuejs.org/) pour le front
[Quasar](https://quasar.dev/) pour des composants déjà prêts

## Docker compose

Mise en place des environnements de travail :
- Php (8.2) + MariaDB (10.5)
- NodeJS (16) + Vue-cli (3) + Quasar-cli

## Serveur

Le serveur (<abbr title="Virtual Private Server">VPS</abbr>) déjà prêt utilise :
- Debian 11
- Nginx
- Php 8.2
- MariaDB 10.5
