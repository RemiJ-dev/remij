# Démarrer un projet Symfony

- Prérequis
- La commande `symfony`
- Démarrer un projet
- Structure d'un projet

## Prérequis

- [PHP](https://www.php.net/manual/fr/install.php)
- [Composer](https://getcomposer.org/doc/00-intro.md)
- [Symfony CLI](https://symfony.com/download)

## La commande `symfony`

- Pour créer une nouvelle application
- Créer un serveur web (avec https)
- Vérifier les vulnérabilités
- Intégration de [Platform.sh](https://symfony.com/cloud/)

## Démarrer un projet

- `symfony new nouveau-projet`
- `symfony new --webapp nouveau-projet`
- `git clone` et `composer install`
