## Description

Bonjour tout le monde !
Aujourd'hui, on démarre un projet Symfony depuis la ligne de commande et je vous présente également la structure d'un projet Symfony, les différents dossiers et les fichiers importants. 

Pour cela, il nous faut avoir au préalable :
- Php en version 8.1 (minimum) (https://www.php.net/manual/fr/install.php)
- Composer (https://getcomposer.org/doc/00-intro.md)
- Symfony CLI (outil en ligne de commande) (https://symfony.com/download)


Attention, petite erreur par habitude : aucun javascript n'est inclus avec l'option webapp dans la version 6 de Symfony. C'était le cas dans la version 5.4 et la version 6.0, mais ça n'est plus le cas.

Projet fil rouge qui sera amélioré tout au long du cours : https://github.com/Drakolab-yt/blog

Dans la prochaine vidéo, je vous présenterai les routes et les controllers, pour enfin afficher des choses ! 

À très bientôt pour la suite de ce cours !
