## Description

Bonjour tout le monde !

Aujourd'hui, dans le cours Symfony, nous faisons en sorte d'avoir des tables dans notre base de données (BdD). Pour cela, on va utiliser le système de migrations, fourni par le bundle DoctrineMigrationsBundle (installé par défaut avec Doctrine). Ces migrations sont des transitions, permettant de refléter les changements du code dans la BdD.

Documentations :
- Doctrine dans Symfony : https://symfony.com/doc/current/doctrine.html
- Celle de Doctrine : https://www.doctrine-project.org/projects/doctrine-orm/en/2.16/index.html
