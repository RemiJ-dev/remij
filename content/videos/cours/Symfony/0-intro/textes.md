## Description

Aujourd'hui, on introduit le futur cours Symfony de la chaîne !
Nous allons étudier Symfony (version 6) depuis zéro, pour vous permettre de créer des sites web sur mesure.

Pour cela, plusieurs notions importantes doivent être connues avant de commencer : 
- Le PHP orienté objet (https://openclassrooms.com/fr/courses/1665806-programmez-en-oriente-objet-en-php)
- l'utilisation de Composer (https://getcomposer.org/)

À très bientôt pour le début de ce cours !

Documentation de Symfony : https://symfony.com/doc/current/index.html
Sylius : https://sylius.com/fr/
