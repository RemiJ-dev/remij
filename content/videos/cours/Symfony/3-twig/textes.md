## Description

Bonjour tout le monde !
Maintenant que nous savons appeler les urls de notre site, il est temps d'afficher du contenu, avec Twig. Nous allons voir comment utiliser le moteur de templates de Symfony pour construire du html (ou d'autres formats). Cette partie concerne la théorie, pour parcourir les principales notions que nous utiliserons dans une seconde vidéo de mise en pratique.

Documentations :
- Twig : https://twig.symfony.com/doc/3.x/
- Symfony sur les templates : https://symfony.com/doc/current/templates.html

Projet fil rouge : https://github.com/Drakolab-yt/blog
La Pull Request liée à la vidéo : https://github.com/Drakolab-yt/blog/pull/3

Dans la prochaine vidéo, nous parlerons de Doctrine, l'ORM (Object Relational Mapper) utilisé par Symfony et donc de base de données.

À très bientôt pour la suite de ce cours !
