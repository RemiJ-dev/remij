## Description

Bonjour tout le monde !
Aujourd'hui, on aborde deux notions centrales, mais indissociables : les routes et les contrôleurs. 

Avant d'entrer dans le vif du sujet, il nous faut bien comprendre ce qu'est le modèle MVC (Modèle Vue Contrôleur) qui va structurer nos projets. En effet, les fichiers vont être rangés suivant ce modèle et cela va nous permettre de rendre notre code plus maintenable et plus lisible.

Documentations Symfony :
- Les routes : https://symfony.com/doc/current/routing.html
- Les controllers : https://symfony.com/doc/current/controller.html

Projet fil rouge : https://github.com/Drakolab-yt/blog

Dans la prochaine vidéo, nous parlerons de Twig, le moteur de templates de Symfony. Cela nous permettra de créer des vues, donc d'afficher du contenu, tout en factorisant nos fichiers HTML (et autres formats de vues).

À très bientôt pour la suite de ce cours !
