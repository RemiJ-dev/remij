## Description

Dans cette vidéo, on va voir comment personnaliser notre profil GitHub simplement, grâce à une astuce de GitHub et au format markdown.

L'astuce est simple : il suffit de créer un repository publique portant le nom de votre profil, avec un fichier README.md à la racine. C'est le contenu de ce fichier qui sera interprété et affiché.

Je vous présente quelques bases de markdown, mais sachez que vous pouvez aller beaucoup plus loin, avec des tableaux, des diagrammes et autres outils avancés. Ici, on ne voit que les bases pour une mise en forme simple, mais la documentation (https://docs.github.com/en/get-started/writing-on-github) vous en dit bien plus.

Exemple sur le profil de Drakolab : https://github.com/Drakolab-yt
En savoir plus sur le markdown pour GitHub : https://docs.github.com/en/get-started/writing-on-github
Des badges à personnaliser : https://shields.io/
